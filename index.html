学习Linux命令

长期更新

由于实验室服务器用的是Linux系统，平时都是用ssh登录管理文件；另外，计算机专业不会Linux那肯定是说不过去的。之前只会一些很基本的linux指令，现在是时候系统地学一遍了。

计划是按照菜鸟教程学，每天学4个，并做好记录。也不求全部掌握吧，对于一些冷门的命令，但至少应该知道这大概是干什么用的。

2022.7.3

cat

cat（英文全拼：concatenate）命令用于连接文件并打印到标准输出设备上。

chattr

chattr命令用于改变文件属性。

chgrp

chgrp（英文全拼：change group）命令用于变更文件或目录的所属群组。

但是我不太懂得群组咋分的。

chmod

chmod（英文全拼：change mode）命令是控制用户对文件的权限的命令。

  • u 表示该文件的拥有者，g 表示与该文件的拥有者属于同一个群体(group)者，o 表示其他以外的人，a 表示这三者皆是。
  • + 表示增加权限、- 表示取消权限、= 表示唯一设定权限。

2022.7.4

chown

chown（英文全拼：change owner）命令用于设置文件所有者和文件关联组的命令。
chown 需要 root 的权限才能执行此命令。

算了，就这样吧

cksum

cksum命令用于检查文件的CRC是否正确。确保文件从一个系统传输到另一个系统的过程中不被损坏。

cmp

cmp 命令用于比较两个文件是否有差异。

当相互比较的两个文件完全一样时，则该指令不会显示任何信息。若发现有所差异，预设会标示出第一个不同之处的字符和列数编号。若不指定任何文件名称或是所给予的文件名为”-“，则cmp指令会从标准输入设备读取数据。

diff

diff 命令用于比较文件的差异。

diff 以逐行的方式，比较文本文件的异同处。如果指定要比较目录，则 diff 会比较目录中相同文件名的文件，但不会比较其中子目录。

2022.7.5

diffstat

diffstat 命令根据diff的比较结果，显示统计数字。

file

file 命令用于辨识文件类型。

find

find 命令用来在指定目录下查找文件。任何位于参数之前的字符串都将被视为欲查找的目录名。如果使用该命令时，不设置任何参数，则 find 命令将在当前目录下查找子目录与文件。并且将查找到的子目录和文件全部进行显示。

git

git 命令是文字模式下的文件管理员。

接受，并为此自豪

今天上了本学期第一节早八，量子物理。昨天运筹学作业把我心态搞崩了，单纯形表算了半天愣是算不出来。气起来今天直接写了个解单纯形表的程序，把问题解决了。看了一下程序输出，发现自己昨天手算的时候第二个表就算错了，淦。

早八上课前就开始写程序了，用iPad带个小键盘写的，结果iPad写程序还勉强能写，调试就比较难搞了，没办法只能先上传到github上。上午3，4节是羽毛球课，这学期的羽毛球课有点无聊，没有对手，但是练网前挑球左一下右一下的居然也挺累。中午睡的挺香的，没定闹钟，一下子睡到了15：22，下午的课差点就迟到了。听说我中午打呼噜巨响，把室友给崩醒了。

今天看完了《人世间》。寒假我就开始跟爸妈一起看这部剧，当时看到周家父母去世的时候我竟也忍不住掉眼泪。这部剧有很多剧情是值得我们反思的，比如对待亲人，对待朋友的方法，比如在遭遇挫折的时候如何认清当下最重要的事物。无论什么时候，人才是最重要的，过分追求功名利禄是会将人异化的。但至于什么是异化呢？“异化”这个词我是在寒假的时候读《穿透》这本书看到的。现在我有了新的体会：“异化”就是被剥夺了感受幸福的能力。当人们开始因为没有得到更多的东西而感受不到幸福的时候，他们就变得越来越不可能再珍视身边的人。所以成功的PUA，也许就是让人们无缘无故降低幸福感的话术。

所以我该怎么做呢？我应该为自己的一切事实感到自豪和幸福。我曾认为自己从小缺少和同龄人的接触导致我现在社交能力不行，甚至觉得是自己的家庭对自己束缚的太严重了。现在想想，我可能确实没什么社交的天赋，但这有什么关系呢？这只是我没能得到的东西之一，不必为此感到失落或遗憾。确实跟我共情的人不是很多，比如说应该没什么人会在博客上写自己的种种思考。但是我为什么不能因此自豪呢？非要遵从别人的价值观来感到稳妥吗？

高傲，沉稳，充满激情。这才是我该有的姿态。

前两天看到了那个北京朝阳区实验小学老师辱骂学生的事，实在是让我有点触目惊心。那两个老师的语气，还有其他学生的起哄，这他妈谁顶的住。难道这就是所谓的更好的教育资源吗？我一向排斥北上广深这些房价巨高的地方，感觉过去就是给资本家打工。掏光积蓄，付个首付，再背上几十年的房贷，最大也只能买个七八十平的房子。有人说北上广深的教育资源是其它任何地方都比不上的，仔细想想，这本末倒置了。人们总是容易忽视人本身的重要性，而追求过多外在的东西。与其进这一所所谓的教育资源丰富的学校，然后低声下气，甚至对于老师都有过分的崇拜，在我看来，还不如像我一样。读个县城的小学，中学，通过农村专项混个985读一读。或许北京的学校出的人才更多，甚至有可能视频里那个趾高气扬的老师对教学颇有经验，指导出了很多优秀的学生，但是对于学生本人来说，最后在学业上成功与否，很大程度上取决于他是否从学习中获得了足够的乐趣。引导孩子对知识的兴趣，对世界的独立思考，这很大程度上是为人父母应该做的，而不是一股劲想要挤进大城市，追求所谓的教育资源，用尽一生还房贷，996给资本家打工。

现在已经是2022年了。今年5，6月份又要进行社团换届了，我也将把羽协传给下一任会长。下学期只有16周，考完之后就要搬宿舍。709里，我读计算机，其他三人一工一管一人文，换宿舍之后，可以说是各奔东西了。现在想想，就已经有点舍不得了。我们宿舍都是通过高考的优惠政策进来的，我和另外两个一样，是农村专项降本一进来，还有一个是国家专项进来的。大家都不是啥大富大贵人家，四个人里面可能我花钱是最不含糊的。我们相处的也挺融洽的，大家都挺淳朴的，很多事情都可以相互包容。应该说，在科大里确实是没见过知乎上说的那些拉帮结派，官僚主义。ljc这人不错，很多地方值得我学习。一方面吧，他学习能力是有的，还挺强。更重要的是，很多事情，他放的下，活的比较快乐，每天晚上都睡得挺好。在这一点上，我就逊色多了。这学期我是最晚考完的，现在宿舍就只剩我一个人。

这个寒假，我还是想学点东西的。或许其实学多少东西不重要。重要的是，我能达到那种在学习中感受到快乐的状态。眼下还是有很多东西可以学的。前两天因为用脚本刷课，被封号了，到现在都还没给我解开。

这不得回去学一学网页前端，计算机网络还有JavaScript的相关知识，写一个更强的脚本出来。希望能够度过一个充实的寒假吧。

大二平日随笔XII

球拍拉线

下周就是第16周了，很多课都结课了，这学期将要步入尾声。有点遗憾，感觉这学期学到的东西不够多。发现自己快一个月没有更新博客了，我已经忙到没有时间闲下来思考一些东西，这不应该。

这三周每周打断一根拍线，就在这周，终于把备用拍的线也打断了。没办法了，今天就骑个小电动车。跑到几公里外去拉线。

复变期末计算错多了，还好期中考了96，但是这样我也痛失4.3，最终4.0收场。这2学分的4.0只让我的gpa提高了0.01。我上学期那些门课的3.0，3.3，让我的gpa足足下降了0.2。我总算亲身体验到，把gpa往上提是多么艰难。

不过这样一来，我产生了一种想法。目前我的gpa虽然捞不到多少奖学金，但是保研还是有希望的。再加上，提高gpa实在太难，太慢了，所以，我似乎可以把gpa看得轻一点。我要追求的是，在学习中真正的感受到快乐，而不是为了gpa去读书。而且很有可能，以这种心态学习，我才会学的好，gpa也会更高。

前两天看到了一个北大的大三学生写的CS自学指南。里面总结了很多计算机相关的课程资源，大多是国外名校的公开课。优秀软件，数学，编程相关，编程无关等等一应俱全。这确确实实的激起了我学习的欲望。由于是国外的课程，从教材到网课全是英文，这也可以顺便锻炼我的英语水平。

我发现我有个缺点是容易受外界评价的影响，没办法保持自己的主见，这是我没有想到。计算机专业的培养方案里，多了“量子物理”这门课。大部分学生对此颇有怨言。我发现看着他们骂骂咧咧，我也变得抵触这门课。不过，我静下心来想，其实自己对这个世界的运行原理有着一种强烈的、本能的好奇。所以当我重新问自己是否想要学这门课的时候，我发现我还是想学的。之前产生厌恶这门课的情绪，很大一部分是源于他人的观点，还有就是gpa降低后，我的好奇心被抑制了，变得畏缩。这得改。

目前还有数电实验大作业，心理学期末论文，数据结构大作业，ICS的一堆lab。纯纯的ddl战士啊。
